Pathfinder Monster Database Help

All queries must be ran against the "monsters" table in the database.

To execute a query using the keyboard, either hit "Alt+Q" or "F5".

Creating the database

To create the database, you must use the .csv version of the monster database from d20pfsrd.

After obtaining the file, click "Create Database...". Select the downloaded file and then select the spot where you would like to keep the database file.
